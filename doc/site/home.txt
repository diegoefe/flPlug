This is the flPlug Wiki!

flPlug is a library whose purpose is to provide a type-safe interface for the
event callback system of the Fast Light ToolKit (FLTK) library.

For an example of code usage click [here](http://sourceforge.net/p/flplug/home/Usage/ "Sample code")
